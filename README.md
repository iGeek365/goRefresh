# goRefresh
#### 使用

项目gradle文件添加
```
allprojects {
          repositories {
              ...
              maven { url "https://jitpack.io" }
          }
      }
 ```
module下添加依赖

   ```
dependencies{
      compile 'com.github.yanyiqun001:goRefresh:0.6'
}
 ```
